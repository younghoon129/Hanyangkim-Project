# MySQL 기본 문법
- DDL(데이터의 기본 구조 및 형식 변경)
    - CREATE
    - DROP
    - ALTER
- DQL(데이터 검색)
    - SELECT
- DML(데이터 조작- 추가, 수정, 삭제)
    - INSERT
    - UPDATE
    - DELETE
- DCL(데이터 및 작업에 대한 사용자 권한 제어)
    - COMMIT
    - ROLLBACK
    - GRANT
    - REVOKE

## DQL
- SELECT
    - SELECT (필드명) FROM (테이블명);
    - 모든 필드 조회 시
        - SELECT (*) FROM (테이블명);
    - Milliseconds 필드는 60000으로 나눠 분 단위 값으로 출력
        - SELECT Milliseconds / 60000 AS '재생 시간/분' FROM tracks;
- ORDER BY(FROM 후에 작성)
    - 오름차순(ASC-기본 값)
    - 내림차순(DESC)
### Filtering
- Clause(특정 기능 수행하도록 지정하는 문장)
    - DISTINCT(SELECT와 같이 작성)
        - 조회 결과에서 중복된 레코드(데이터) 제거
        - SELECT DISTINCT (필드명)
    - WHERE(FROM 후에 작성)
        - 조회 시 특정 검색 조건을 지정
        - 비교 및 논리 연산자 사용 구문
        - WHERE (필드명) = '레코드(데이터)';
        - WHERE (필드명) BETWEEN 레코드(데이터(숫자)) AND 레코드(데이터(숫자)) ORDER BY (필드명);
        - WHERE (필드명) IN (NOT IN) ('레코드(데이터)', '레코드(데이터)', '레코드(데이터)');
        - WHERE (필드명) LIKE '%레코드(데이터)'; -> 레코드(데이터)로 끝나는 데이터 조회
        - WHERE (필드명) LIKE '___a'; -> 4자리면서 'a'로 끝나는 데이터 조회
    - LIMIT
        - LIMIT [offset(0 또는 양의 정수로 하나 혹은 두개 인자 사용), ] (조회할 데이터 갯수);

- Operator(조건 비교, 데이터 선택)
    - BETWEEN
    - IN
    - LIKE
    - Comparison
    - Logical

## Grouping data
- GROUP BY
    - FROM, WHERE 절 뒤에 배치 
    - 그룹화 할 필드 목록 작성
- Aggregation Functions
    - 집계 함수
    - 값에 대한 계산을 수행하고 단일한 값을 반환하는 함수
